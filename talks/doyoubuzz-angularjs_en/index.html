Plusieurs possibilités pour le rendu :
  • tout le rendu se fait côté serveur
    -> peu de flexibilité
  • tout le rendu se fait côté client
    -> pas de JS en mode vue

    -> maintenabilité 0
  • le client et le serveur utilisent le même moteur de rendu
    -> pas encore vraiment possible
  • le serveur rend des morceaux du CV et le client les place ✓
