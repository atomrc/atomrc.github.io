+ : Problem

Many possibilities
  • all the rendering on server side
    -> no flexibility
  • all the rendering on client side
    -> no JS on 'view' mode
  • duplication of rendering logic between server and client
    -> sustenability 0
  • the client and the server share the same rendering engine (twig.js)
    -> not really possible right now
  • the server render all the views and the client is only placing them✓
